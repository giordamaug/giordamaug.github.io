Installation¶

Requiremensts¶

HELP depends on the following packages:

  • numpy,

  • pandas,

  • typing,

  • matplotlib,

  • skimage,

  • sklearn,

  • tqdm,

  • lightgbm,

  • tabulate,

  • networkx,

  • karateclub,

  • ipywidgets,

  • supervenn.

Pip¶

HELP allows pip-based installation, please use the following command:

pip install git+https://github.com/giordamaug/HELP.git
