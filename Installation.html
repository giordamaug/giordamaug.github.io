Installation

HELP depends on the following packages: numpy, tqdm, typing, pandas, supervenn, matplotlib, ipywidgets, scipy, sklearn

Pip

HELP allows pip-based installation, please use the following command:

pip install help

Nightly

In case you want to experiment with the latest TorchEEG features which are not fully released yet, please run the following command to install from the main branch on github:

pip install git+https://github.com/giordamaug/HELP.git
